Function array_clean

Cleans an array. It filters elements, removes duplicate values and reorders the keys.

Elements will be filtered with the array_filter() function. If$callback is null, all entries of array equal to FALSE will be removed.

The keys will be re-ordered with the `array_values() function only if all the keys in the original array are numeric.
